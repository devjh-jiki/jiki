# Learning Skills

> 한국어: [README.ko.md](./README.ko.md)

Coach-style skills that help you learn AI and technical knowledge while developing.
Instead of handing you a summary, they make you form hypotheses, verify them against evidence, and turn learning into small artifacts.

**Model-invoked** (the agent uses it automatically in a learning context)

- [open-source-reverse-engineering-coach](./open-source-reverse-engineering-coach/SKILL.md) — A coach for learning architecture, interfaces, tests, and trade-offs by reverse-engineering open source projects
- [technical-book-coach](./technical-book-coach/SKILL.md) — Learn technical books, essays, and docs through coaching (when English text is pasted, separates Korean translation + coaching)

## Shared philosophy

- Narrow down to a small slice instead of explaining the whole thing.
- Ask the user for their hypothesis/understanding first, then verify against evidence (code, tests, docs).
- Distinguish facts backed by code from inferences.
- End each session with a small learning artifact (saved only after the user approves).
