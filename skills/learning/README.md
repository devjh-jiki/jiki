# Learning Skills

개발하면서 AI·기술 지식을 학습하는 과정을 돕는 코치형 스킬.
요약을 던져주는 대신, 가설을 세우게 하고 근거로 검증하며 학습을 작은 산출물로 바꾼다.

**Model-invoked** (학습 맥락이면 에이전트가 자동 사용)

- [open-source-reverse-engineering-coach](./open-source-reverse-engineering-coach/SKILL.md) — 오픈소스 프로젝트를 역공학하며 아키텍처·인터페이스·테스트·트레이드오프를 배우는 코치
- [technical-book-coach](./technical-book-coach/SKILL.md) — 기술 서적·에세이·문서를 코칭으로 학습 (영문 붙여넣기 시 한글 번역 + 코칭 분리)

## 공통 철학

- 통째로 설명하지 않고 작은 슬라이스로 좁힌다.
- 사용자에게 먼저 가설/이해를 묻고, 근거(코드·테스트·문서)로 검증한다.
- 코드로 뒷받침되는 사실과 추론을 구분한다.
- 세션을 작은 학습 아티팩트로 마무리한다 (저장은 사용자 승인 후).
